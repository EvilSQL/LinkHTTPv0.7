
Список изменений программы LinkHTTP

v 0.4
1. Добавлена загрузка всех новых сертификатов ФСРАР (с оригиналных ресурсов)
2. Добавлена библиотека для работы с архивами zip, rar, gz, tar
3. Исправлены мелкие ошибки.

v 0.3
1. Добавлена многопоточность при индексирования настроечных файлов.
2. Переиндексирование настроечных файлов происходит через заданное администратором время (по умолч. 15 минут)
3. Добавлен сборщик мусора (очистка не используемых настроечных файлов через заданное время (по умолч. 1 час) освобождение памяти, очистка мусра в консоли)
4. Изменена коллекция Dictonary на ConcurrentDictionary для многопоточного доступа.
5. Переиндексирование происходит только если в папке с настрочеными файлами были изменения (например добавлен файл)
6. Добавлена иконка для программы.
7. Добавлена проверка ИНН.
8. Проведена оптимизация.
9. Улучшена безопастность сервера.

v 0.2 

1. Переписана функция получения настроечного файла (теперь для получения настроечных файлов исползуется отдельный поток который не мешает получению других файлов)
2. Добавлена возможность использовать сервер для получения настроечных файлов в случае сбоя программы LinkService либо при переносе на другой компьютер.
3. Добавлено переиндексирование настроечных файлов.
4. Реализована обработка ошибок, ведение лог файлов.
5. Изменена платформа с NET.2.0 на NET. 4.0
6. Устранены мелкие ошибки.
7. Оптимизирована многопоточность серверной части: 32 потока на каждое ядро при максимальной нагрузке и 16 на каждое ядро в обычном режиме.

v 0.1

1. Добавлена многопоточность серверной части
2. Последнии настроечные файлы можно получить указав в запросе ИНН и отпечаток сертификата.
3. Первая версия программы.
